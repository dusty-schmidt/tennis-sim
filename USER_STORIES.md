# 📋 User Stories & Roadmap

## ✅ Completed Stories
- **As a DFS Player**, I want to see the 90th percentile (ceiling) of a player so I can win high-stakes GPP tournaments.
- **As a Sharp Bettor**, I want matches simulated on specific surfaces (clay/hard) so I can find edge against bookies.
- **As a Developer**, I want a single command setup (Docker) so I can start working on any machine instantly.
- **As a User**, I want to know player ownership based on real-world constraints like 'Min Uniques'.

## 🔜 Planned Stories
- **As a Pro User**, I want to export my 150 optimal lineups directly to a CSV for DraftKings upload.
- **As a User**, I want to 'Late Swap' my lineups if a match is delayed or a player withdraws.
- **As a Data Scientist**, I want to see calibration charts comparing simulation accuracy against actual historical results.
